# Assets Directory

This directory contains images and media for the Canvas Connector documentation.

## Files

- **favicon.ico** - Site favicon (browser tab icon)
- **favicon.png** - High-resolution logo for header
- **banner.png** - Project banner for README and homepage

## Usage in Documentation

Reference images in your markdown files like:

```markdown
![Canvas Connector](banner.png)
```

Or in HTML:

```html
<img src="banner.png" alt="Canvas Connector Banner" />
```
